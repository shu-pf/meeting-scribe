# MeetingScribe

Mac用のローカル会議議事録作成アプリケーション

## 概要

MeetingScribeは、Macのステータスメニューに常駐し、会議の画面録画から文字起こし、要約までを自動で行うアプリケーションです。すべての処理をローカルで完結させることで、セキュアに会議内容を記録できます。

### 主な特徴

- **ステータスメニュー常駐**: いつでもワンクリックで録画を開始
- **画面録画**: 選択したウィンドウまたは画面全体を録画・録音
- **自動文字起こし**: Whisperを使用したローカル文字起こし
- **自動要約**: ローカルLLM（Ollama等）による会議要約
- **完全ローカル処理**: クラウド連携不要でセキュア

## 機能

### 画面録画
- ウィンドウ単位または画面全体の録画に対応
- 音声も同時にキャプチャ
- ステータスメニューから簡単に開始・停止

### 文字起こし（Whisper）
- アプリケーションに組み込み済み
- 初回起動時にモデルを選択してダウンロード
- 録画終了後に自動実行

### 会議要約（ローカルLLM）
- Ollama等のローカルLLMを利用
- 文字起こし完了後に自動実行
- モデルは設定画面で選択可能

## システム要件

- macOS
- Ollama（要約機能を使用する場合）

## インストール

[https://shu-pf.github.io/meeting-scribe/](https://shu-pf.github.io/meeting-scribe/) にアクセスし、「ダウンロード」ボタンをクリックしてください。

## 使い方

1. アプリケーションを起動すると、ステータスメニューにアイコンが表示されます
2. 初回起動時は、Whisperモデルのダウンロードを行います
3. ステータスメニューのアイコンをクリックして、録画を開始
4. 録画するウィンドウを選択
5. 録画を停止すると、自動的に文字起こしと要約が実行されます
6. 結果は設定した出力フォルダに保存されます

## 設定

アプリケーションをクリックすると設定画面が開きます。

- **文字起こしモデル**: 使用するWhisperモデルの選択
- **要約モデル**: 使用するローカルLLMモデルの選択
- **出力フォルダ**: 録画・文字起こし・要約の保存先
